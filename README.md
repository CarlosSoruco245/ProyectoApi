# ProyectoApi

A comprehensive orders management system featuring a backend API, database, and frontend application.

## 📁 Project Structure

This project is organized into three main components:

### 1. **PedidosApi** - Backend API
The REST API server that handles business logic and data operations for the orders system.

- Built to manage orders and related operations
- Provides endpoints for CRUD operations
- Handles request validation and response formatting

### 2. **BD** - Database
Database configuration and schema definitions.

- Stores orders data and related information
- Contains database initialization scripts

### 3. **frontend-dbpedidos** - Frontend Application
User interface for interacting with the orders management system.

- Modern web interface
- Integrates with the PedidosApi backend
- Responsive design for various devices

## 🚀 Getting Started

### Prerequisites
- Node.js or your framework runtime
- Database server (as per BD configuration)
- Package manager (npm, pip, etc.)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/CarlosSoruco245/ProyectoApi.git
   cd ProyectoApi
   ```

2. **Install Backend Dependencies**
   ```bash
   cd PedidosApi
   npm install  # or appropriate package manager
   ```

3. **Setup Database**
   ```bash
   cd ../BD
   # Run database initialization scripts
   ```

4. **Install Frontend Dependencies**
   ```bash
   cd ../frontend-dbpedidos
   npm install  # or appropriate package manager
   ```

### Running the Project

#### Backend API
```bash
cd PedidosApi
npm start
```

#### Frontend
```bash
cd frontend-dbpedidos
npm start
```
